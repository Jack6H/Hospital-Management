## 安装与设置指南
1. 安装 Node.js:
   ```
  访问 https://nodejs.org/zh-cn/download 下载并安装 Node.js。
   ```
2. 克隆仓库:
   ```
   git clone https://github.com/yazdanhaider/Hospital-Management.git
   ```
3. 进入后端项目目录:
   ```
   cd Hospital-Management Backend
   ```
4. 安装后端依赖:
   ```
   npm install
   ```
5. 启动后端服务器:
   ```
   npm run dev
   ``` 
6. 进入前端项目目录（在新终端中执行）:
   ```
   cd Hospital-Management Frontend
   ```
7. 安装前端依赖:
   ```
   npm install
   ```
8. 启动前端开发服务器:
   ```
   npm run dev
   ```
